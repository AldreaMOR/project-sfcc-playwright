# project-sfcc-playwright
Project with QA Automation in Salesforce Commerce

## Steps to create project with QA Automation:
1# Create a project to use typescript and execution tests in salesforce cloud project, on github
2# Clone this project in PC
3# Install the playwright dependencies with command
   - Run the command in the root folder of the project ~ npm init playwright@latest
   - Select typescript
   - Select tests location tests
   - Select test runner playwright test
   - Select browsers chromium, firefox and webkit
   - Select yes to install dependencies
   - Select no to install example tests
   - Select yes to install git ignore
4# Create your tests in the folder tests
5# Execute your tests with the command npm run test
